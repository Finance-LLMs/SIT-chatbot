# SIT Chatbot

## How to run the code

1. Clone this repo
    ```bash
    git clone https://github.com/Finance-LLMs/SIT-chatbot.git
    ```

2. Setup the environment variables
    ```bash
    vim .env        # Ubuntu
    notepad .env    # Windows
    ```

3. Install the dependencies

    ```bash
    npm install                          # Frontend
    pip install -r requirements.txt      # Backend
    ```

4. Run the script

    ```bash
    npm start
    ```

5. Visit localhost

    ```bash
    http://localhost:3000/
    ```
